## 🎮 프로젝트 소개

이 프로젝트는 raylib 라이브러리를 활용하여 C 언어로 구현한 전통적인 스네이크 게임입니다. 
SOLID 원칙을 준수하여 모델, 로직, 뷰, 컨트롤러가 분리된 아키텍처로 설계되었습니다.

## ✨ 주요 기능

- [ ] 뱀이 화면을 이동하며 음식을 먹는 기본 게임플레이
- [ ] 음식을 먹을 때마다 뱀의 길이가 증가
- [ ] 벽이나 자신의 몸에 부딪히면 게임 오버
- [ ] 점수 시스템
- [ ] 게임 오버 시 재시작 기능
- [ ] 부드러운 애니메이션 및 그래픽

## 🔧 요구사항

### 필수 소프트웨어

- **C 컴파일러**: MinGW-w64 (Windows 환경)
- **빌드 도구**: Make
- **라이브러리**: raylib
- **운영체제**: Windows 10 이상

### raylib 설치

raylib은 다음 방법 중 하나로 설치할 수 있습니다:

1. **소스에서 빌드** (권장)
   ```bash
   git clone https://github.com/raysan5/raylib.git
   cd raylib/src
   make PLATFORM=PLATFORM_DESKTOP
   ```

2. **미리 컴파일된 라이브러리 사용**
   - [raylib 릴리스 페이지](https://github.com/raysan5/raylib/releases)에서 다운로드

## 🏗️ 빌드 방법

1. 프로젝트 디렉토리로 이동:
   ```bash
   cd snake-game
   ```

2. Make를 사용하여 빌드:
   ```bash
   make
   ```

3. 빌드된 실행 파일은 `bin/` 디렉토리에 생성됩니다.

### 수동 빌드

Makefile이 없는 경우, 다음 명령어로 직접 빌드할 수 있습니다:

```bash
gcc -o snake.exe src/*.c -Iinclude -Llib -lraylib -lopengl32 -lgdi32 -lwinmm
```

## 🚀 실행 방법

빌드가 완료되면 다음 명령어로 게임을 실행합니다:

```bash
./bin/snake.exe
```

또는 Windows 탐색기에서 `bin/snake.exe`를 더블클릭하여 실행할 수 있습니다.

## 🎯 게임 조작법

| 키 | 동작 |
|---|---|
| `↑` (위쪽 화살표) | 뱀을 위로 이동 |
| `↓` (아래쪽 화살표) | 뱀을 아래로 이동 |
| `←` (왼쪽 화살표) | 뱀을 왼쪽으로 이동 |
| `→` (오른쪽 화살표) | 뱀을 오른쪽으로 이동 |
| `R` | 게임 재시작 (게임 오버 시) |
| `ESC` | 게임 종료 |

## 📁 프로젝트 구조

```
snake-game/
├── src/              # 소스 코드
│   ├── main.c       # 메인 진입점
│   ├── game.c       # 게임 로직
│   ├── snake.c      # 뱀 모델
│   ├── food.c       # 음식 모델
│   └── renderer.c   # 렌더링 로직
├── include/         # 헤더 파일
│   ├── game.h
│   ├── snake.h
│   ├── food.h
│   └── renderer.h
├── lib/             # 라이브러리 파일 (raylib 등)
├── bin/             # 빌드 출력 디렉토리
├── Makefile         # 빌드 스크립트
└── README.md        # 프로젝트 문서
```
