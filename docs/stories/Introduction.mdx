# AI Agent 교육 커리큘럼

> **Cursor**와 **Claude Code**를 활용한 AI 기반 개발 실습 가이드

---

## 과정 개요

| 항목 | 내용 |
|------|------|
| **총 교육 시간** | 5시간 |
| **실습 프로젝트** | Snake Game (C, raylib) |
| **GitHub 저장소** | [https://github.com/jhl-labs/snake-game](https://github.com/jhl-labs/snake-game) |
| **필요 환경** | Windows / Linux |
| **대상** | AI Agent 활용에 관심 있는 개발자 |

---

## 학습 목표

이 과정을 완료하면 다음을 할 수 있습니다:

1. **AI Agent의 동작 원리 이해** - LLM 기반 코딩 어시스턴트가 어떻게 작동하는지 파악
2. **Rule 파일 작성** - 프로젝트별 코딩 규칙을 Agent에게 학습시키기
3. **Command 파일 활용** - 반복 작업을 자동화하는 커맨드 정의
4. **Subagent 활용** - 복잡한 작업을 분업하여 처리
5. **MCP 활용** - 외부 도구와 데이터 소스를 Agent에 연결
6. **CI/CD 통합** - GitHub Actions로 Claude Code 자동화
7. **실전 활용** - 테스트 케이스 작성, 버그 수정에 Agent 활용

### Snake Game

**GitHub 저장소:** [https://github.com/jhl-labs/snake-game](https://github.com/jhl-labs/snake-game)

**기술 스택:**
- **언어**: C (C11 표준)
- **그래픽**: raylib
- **빌드**: Make (MinGW-w64)
- **아키텍처**: MVC 패턴

**프로젝트 구조:**
```
snake-game/
├── src/                # 소스 코드
│   ├── main.c          # 진입점, 게임 루프
│   ├── game.c          # 게임 로직
│   ├── snake.c         # 뱀 모델
│   ├── food.c          # 음식 모델
│   └── renderer.c      # 렌더링 (raylib)
├── include/            # 헤더 파일
├── .cursor/            # Cursor 설정
│   ├── rules/          # 코딩 규칙
│   └── commands/       # 커맨드
├── .claude/            # Claude Code 설정
│   ├── rules/          # 코딩 규칙
│   ├── commands/       # 슬래시 커맨드
│   └── skills/         # 에이전트 스킬
├── exercises/          # 실습 파일
├── CLAUDE.md           # 프로젝트 가이드
└── Makefile
```

## 시작하기

### 1. 저장소 클론

실습을 시작하기 전에 GitHub 저장소를 클론하세요:

```bash
git clone https://github.com/jhl-labs/snake-game.git
cd snake-game
```

왼쪽 사이드바에서 **Chapters > 01. 소개 및 환경 설정**을 클릭하여 학습을 시작하세요.
