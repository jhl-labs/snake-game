# 4.2 Claude Code 슬래시 커맨드

Claude Code는 **`/` 슬래시 커맨드**로 빠르게 작업을 실행할 수 있습니다.


![](015-command-claude.png)

---

## 내장 슬래시 커맨드

| 커맨드 | 설명 |
|--------|------|
| `/help` | 도움말 및 사용 가능한 커맨드 목록 |
| `/clear` | 대화 히스토리 초기화 |
| `/compact` | 컨텍스트 압축 (긴 대화 요약) |
| `/init` | CLAUDE.md 파일 생성 |


---

## /help - 도움말

<Source
  language="bash"
  code={`
claude
> /help

# 출력 예시:
Available commands:
  /help     - Show this help message
  /clear    - Clear conversation history
  /compact  - Compress context
  /init     - Initialize CLAUDE.md
  ...
`}
/>

---

## /clear - 대화 초기화

새로운 작업을 시작할 때 이전 대화 컨텍스트를 **완전히 초기화**합니다.

<Source
  language="bash"
  code={`
> /clear

# 모든 대화 히스토리가 삭제됨
# 새로운 세션 시작
`}
/>

**사용 시점:**
- 완전히 다른 작업 시작
- 이전 대화가 현재 작업에 방해될 때
- 깨끗한 상태에서 테스트하고 싶을 때

---

## /compact - 컨텍스트 압축

긴 대화를 **요약하여 컨텍스트 공간** 확보합니다.

<Source
  language="bash"
  code={`
> /compact

# 이전 대화가 요약됨:
# "사용자가 snake.c의 버그를 수정하고 테스트를 추가함"
`}
/>


**사용 시점:**
- 대화가 길어져서 응답이 느려질 때
- "컨텍스트 한도 초과" 경고가 나올 때
- 핵심 정보만 유지하고 싶을 때

---

## /init - CLAUDE.md 생성

프로젝트에 맞는 **CLAUDE.md 파일을 자동 생성**합니다.

<Source
  language="bash"
  code={`
> /init

# Agent가 프로젝트 구조를 분석하고
# CLAUDE.md 파일을 자동 생성
`}
/>


---

## 커스텀 슬래시 커맨드

Claude Code는 `.claude/commands/` 폴더에서 **커스텀 커맨드**를 정의할 수 있습니다.

### 파일 위치

**위치:** `.claude/commands/*.md`

### 구조

<Source
  language="markdown"
  code={`
---
name: build
description: 프로젝트를 빌드합니다
---

프로젝트를 빌드해주세요.

\`mingw32-make\` 명령어를 사용합니다.
빌드 결과를 확인하고 에러가 있으면 알려주세요.
`}
/>

### 프로젝트의 커스텀 커맨드

| 파일명 | 커맨드 | 설명 |
|--------|--------|------|
| `build.md` | `/build` | 프로젝트 빌드 |
| `run.md` | `/run` | 실행 |
| `test.md` | `/test` | 테스트 실행 |
| `clean.md` | `/clean` | 빌드 정리 |
| `analyze.md` | `/analyze` | 코드 분석 |

---

## 슬래시 커맨드 사용 예시

### /build

<Source
  language="bash"
  code={`

`}
/>
