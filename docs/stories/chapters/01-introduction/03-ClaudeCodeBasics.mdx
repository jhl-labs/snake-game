# 1.3 Claude Code 기본 사용법

---

## Claude Code란?

Claude Code는 **터미널에서 실행되는 에이전틱 코딩 도구**입니다.
자연어로 요청하면 파일을 직접 편집하고, 명령어를 실행하며, Git 워크플로우를 처리합니다.

<Mermaid chart={`
flowchart LR
    subgraph Terminal["터미널 환경"]
        Input["자연어 요청"]
        Agent["Claude Agent"]
        Tools["도구 실행"]
        Output["결과"]
    end

    Input --> Agent
    Agent --> Tools
    Tools -->|"파일 편집"| File["📄 Edit"]
    Tools -->|"명령 실행"| Bash["⚡ Bash"]
    Tools -->|"코드 검색"| Search["🔍 Search"]
    Tools --> Output
`} />

## 기본 대화

프롬프트에 자연어로 요청을 입력합니다:

<Source
  language="text"
  code={`
> 이 프로젝트의 구조를 설명해줘

> snake.c에서 뱀이 이동하는 함수를 찾아줘

> game_update 함수에서 충돌 감지가 어떻게 되는지 분석해줘
`}
/>

### Agent 동작 관찰

Claude Code는 요청을 처리하면서 **사용하는 도구를 실시간으로 표시**합니다:

<Source
  language="text"
  code={`
> snake_move 함수를 찾아서 설명해줘

◐ Searching for snake_move in codebase...
  Read src/snake.c (lines 45-80)

snake_move 함수는 뱀의 위치를 업데이트하는 함수입니다...
`}
/>

## 주요 기능

### 1. 기능 개발 (Build Features)

만들고 싶은 것을 설명하면 Claude가 **계획을 세우고 코드를 작성**합니다:

<Source
  language="text"
  code={`
> 새로운 파워업 시스템을 추가해줘.
  뱀이 파워업을 먹으면 일정 시간 동안 속도가 빨라져야 해.
`}
/>

### 2. 디버깅 및 버그 수정

버그나 에러 메시지를 설명하면 **코드베이스를 분석하고 자동으로 수정**합니다:

<Source
  language="text"
  code={`
> make 빌드할 때 "undefined reference to snake_grow" 에러가 나는데 수정해줘

> 뱀이 벽을 통과하는 버그가 있어. 찾아서 고쳐줘
`}
/>

### 3. 코드베이스 탐색

전체 프로젝트 구조를 인식하고 필요한 정보를 찾습니다:

<Source
  language="text"
  code={`
> 충돌 감지 관련 코드가 어디에 있어?

> MVC 패턴에서 각 레이어가 어떤 파일들인지 정리해줘
`}
/>

### 4. Git 워크플로우

커밋, PR 생성 등 Git 작업을 자동화합니다:

<Source
  language="text"
  code={`
> 변경사항을 커밋해줘

> 이 브랜치로 PR을 만들어줘
`}
/>

## 슬래시 커맨드

`/`로 시작하는 특수 명령어를 사용할 수 있습니다:

![](011-claude.png)

| 커맨드 | 설명 |
|--------|------|
| `/help` | 도움말 및 사용 가능한 커맨드 목록 |
| `/clear` | 대화 히스토리 초기화 |
| `/compact` | 컨텍스트 압축 (긴 대화 시 유용) |
| `/init` | CLAUDE.md 파일 생성 |
| `/model` | 사용 중인 모델 확인/변경 |
| `/bug` | 버그 리포트 제출 |

<Source
  language="text"
  code={`
> /help
# 사용 가능한 명령어 목록 표시

> /init
# 프로젝트 루트에 CLAUDE.md 파일 생성
`}
/>

## CLI 옵션

### 프롬프트와 함께 실행 (-p)

<Source
  language="bash"
  code={`
# 단일 작업 실행
claude -p "이 프로젝트의 TODO 항목을 찾아줘"

# 출력만 받기
claude -p "snake.c의 함수 목록" --print
`}
/>

### 파이프라인 연동

Unix 철학에 따라 다른 명령어와 연결할 수 있습니다:

<Source
  language="bash"
  code={`
# 로그 분석
tail -f app.log | claude -p "에러 패턴을 분석해줘"

# 파일 내용 전달
cat error.txt | claude -p "이 에러를 분석해줘"
`}
/>

## Cursor vs Claude Code 비교

| 작업 | Cursor | Claude Code |
|------|--------|-------------|
| 코드 질문 | `Cmd+I` Agent | 프롬프트에 입력 |
| 파일 참조 | `@파일명` | 자동 탐색 또는 파일명 언급 |
| 인라인 수정 | `Cmd+K` | "~에서 ~를 수정해줘" |
| 멀티파일 작업 | Agent | 자연어로 요청 |
| 빌드/실행 | 터미널에서 직접 | "빌드해줘" 요청 |
| 파이프라인 연동 | 불가 | `claude -p` |
