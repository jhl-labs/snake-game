# 5.2 agent.md 작성



##  AGENTS.md (오픈 표준)

### 개요

AGENTS.md는 `robots.txt`의 AI 버전입니다. 프로젝트 루트에 위치하며, 모든 AI 코딩 에이전트가 읽을 수 있는 범용 컨텍스트 파일입니다.

### 지원 도구

- Cursor
- GitHub Copilot
- Zed
- AMP
- Roo Code
- Claude Code (심볼릭 링크 필요)

### 파일 구조

```
project/
├── AGENTS.md           # 메인 컨텍스트 파일
├── CLAUDE.md → AGENTS.md  # 심볼릭 링크 (선택)
├── backend/
│   └── AGENTS.md       # 백엔드 전용 지침
└── frontend/
    └── AGENTS.md       # 프론트엔드 전용 지침
```

### 예시

```markdown
# AGENTS.md

This file provides guidance to AI agents when working with code in this repository.

## 프로젝트 개요
- 기술 스택: Python 3.11, FastAPI, PostgreSQL
- 아키텍처: 마이크로서비스

## 빌드 명령어
- 설치: `pip install -e .`
- 테스트: `pytest tests/`
- 린트: `ruff check .`

## 코드 컨벤션
- docstring: Google 스타일
- 타입 힌트 필수
- 함수 최대 50줄

## 금지 사항
- print() 디버깅 금지
- *args, **kwargs 남용 금지
```

### Claude Code 호환 설정

```bash
# 심볼릭 링크 생성
ln -s AGENTS.md CLAUDE.md

# .gitignore에 추가
echo "CLAUDE.md" >> .gitignore
```
