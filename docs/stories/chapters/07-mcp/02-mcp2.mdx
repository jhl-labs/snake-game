## 1. Context7 - 최신 라이브러리 문서 검색

### Context7이란?

**Context7**은 npm 패키지, 라이브러리의 **최신 문서를 실시간으로 검색**해주는 MCP 서버입니다.
AI가 오래된 정보가 아닌 최신 API 문서를 참조하여 정확한 코드를 생성할 수 있게 해줍니다.

### 왜 필요한가?

| 문제 | Context7 해결책 |
|------|----------------|
| AI의 학습 데이터가 오래됨 | 실시간 최신 문서 검색 |
| Deprecated API 사용 | 현재 버전 API 참조 |
| 버전별 차이 모름 | 특정 버전 문서 조회 |

### 설정 방법

<Source
  language="json"
  code={`
// ~/.claude.json 또는 .cursor/mcp.json
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@anthropic/context7-mcp"]
    }
  }
}
`}
/>

### 활용 예시

<Source
  language="text"
  code={`
> React 19의 새로운 use() hook 사용법을 알려줘

> Next.js 15에서 Server Actions 설정하는 방법을 찾아줘

> raylib 5.0의 새로운 함수들을 확인해줘

# Context7이 최신 공식 문서에서 정보를 가져와
# 정확한 코드 예제와 함께 설명
`}
/>

### 실전 시나리오

<Source
  language="text"
  code={`
# 시나리오: 새 프로젝트에서 최신 라이브러리 사용

> Tailwind CSS v4의 새로운 설정 방식으로 프로젝트를 세팅해줘

# AI 응답:
# 1. Context7로 Tailwind v4 공식 문서 검색
# 2. 최신 설정 파일 구조 확인
# 3. 정확한 설정 코드 생성
`}

## 2. GitHub - 저장소 및 이슈 관리

### GitHub MCP란?

**GitHub MCP 서버**는 저장소, 이슈, PR, 코드 검색 등 GitHub의 모든 기능에 AI가 접근할 수 있게 해줍니다.

### 설정 방법

<Source
  language="json"
  code={`
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "ghp_xxxxxxxxxxxxxxxxxxxx"
      }
    }
  }
}
`}
/>

> **GitHub Token 발급**: Settings > Developer settings > Personal access tokens > Generate new token

### 활용 예시

#### 저장소 관리

<Source
  language="text"
  code={`
> jhl-labs/snake-game 저장소의 최근 커밋 10개를 보여줘

> 이 저장소의 브랜치 목록과 각 브랜치의 마지막 커밋을 알려줘

> main 브랜치와 feature/power-up 브랜치의 차이점을 분석해줘
`}
/>

#### 이슈 관리

<Source
  language="text"
  code={`
> 열린 이슈 목록을 우선순위별로 정리해줘

> "버그: 뱀이 벽을 통과함"이라는 제목으로 이슈를 생성해줘

> #42 이슈에 "수정 완료했습니다. PR #45를 확인해주세요"라고 코멘트 달아줘
`}
/>

#### PR 관리

<Source
  language="text"
  code={`
> 열린 PR 목록과 각각의 리뷰 상태를 보여줘

> PR #45의 변경 사항을 분석하고 코드 리뷰해줘

> 이 변경사항으로 PR을 생성해줘. 제목은 "feat: 파워업 시스템 추가"

/>

### 실전 시나리오

> PR을 생성하고 이슈에 링크해줘

# AI 작업 흐름:
# 1. #123 이슈 내용 확인 (GitHub MCP)
# 2. 관련 코드 검색 및 분석
# 3. 버그 수정
# 4. PR 생성 (GitHub MCP)
# 5. 이슈에 PR 링크 코멘트 (GitHub MCP)
`}
/>

## 6. Database - PostgreSQL/SQLite

### Database MCP란?

**Database MCP 서버**는 AI가 데이터베이스에 직접 쿼리를 실행하고,
스키마를 분석하며, 데이터 기반 의사결정을 지원합니다.

### 설정 방법 (PostgreSQL)

<Source
  language="json"
  code={`
{
  "mcpServers": {
    "postgres": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres"],
      "env": {
        "POSTGRES_CONNECTION_STRING": "postgresql://user:pass@localhost:5432/mydb"
      }
    }
  }
}
`}
/>

### 설정 방법 (SQLite)

<Source
  language="json"
  code={`
{
  "mcpServers": {
    "sqlite": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-sqlite",
        "--db-path",
        "./data/game.db"
      ]
    }
  }
}
`}
/>

### 활용 예시

<Source
  language="text"
  code={`

> users 테이블과 scores 테이블을 조인해서 사용자별 평균 점수를 계산해줘

> 쿼리 성능이 느린 것 같은데, EXPLAIN ANALYZE 결과를 분석하고 인덱스를 제안해줘
