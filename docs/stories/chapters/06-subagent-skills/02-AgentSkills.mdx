# 6.2 Agent Skills

- 에이전트 스킬은 AI 에이전트를 특화된 기능으로 확장하기 위한 오픈 표준입니다.
- Skills는 Agent가 **특정 작업을 수행하는 능력**입니다.
- Claude Code는 `.claude/skills/` 폴더에서 커스텀 스킬을 정의할 수 있습니다.
- cursor는 ~/.cursor/skills 에 정의하면 되지만, ~/.claude에 있는 내용도 사용 가능합니다.  

![](017-skill.png)

---

## 주요 Skills

| 스킬 | 설명 | 활용 예 |
|------|------|--------|
| **코드 리뷰** | 코드 품질 분석 | PR 리뷰, 코드 검토 |
| **버그 수정** | 문제 식별 및 해결 | 에러 분석, 패치 |
| **리팩토링** | 코드 구조 개선 | 중복 제거, 모듈화 |
| **테스트 작성** | 유닛 테스트 생성 | TC 자동 생성 |
| **문서화** | 주석 및 문서 작성 | API 문서, README |

## Skills 파일 구조

**위치:** `.claude/skills/[skill-name]/SKILL.md`

### 예시: code-review 스킬

<Source
  language="markdown"
  code={`
# Code Review Skill

## 목적
코드 변경 사항을 분석하고 품질 피드백을 제공합니다.

## 검토 항목

### 1. 코딩 스타일
- 네이밍 규칙 준수
- 포매팅 일관성
- 주석 품질

### 2. 버그 가능성
- NULL 포인터 체크
- 배열 경계 검사
- 메모리 누수

### 3. 성능
- 불필요한 반복
- 비효율적인 알고리즘
- 메모리 사용량

### 4. 보안
- 입력 검증
- 버퍼 오버플로우
- 하드코딩된 값

## 출력 형식
- 심각도: Critical / Warning / Info
- 위치: 파일:라인
- 설명: 문제점과 해결책
`}
/>

## Skills 활용 예시

### 코드 리뷰 요청

<Source
  language="text"
  code={`
> snake.c의 최근 변경 사항을 리뷰해줘

Agent (code-review skill 활용):
"코드 리뷰 결과입니다:

[Warning] snake.c:45
- NULL 체크 누락
- 권장: if (p_snake == NULL) return false;

[Info] snake.c:52
- 매직 넘버 사용
- 권장: #define MAX_SPEED 10 사용"
`}
/>

### 테스트 생성 요청

<Source
  language="text"
  code={`

Agent (test-generation skill 활용):

1. test_snake_move_up - 위로 이동
2. test_snake_move_boundary - 경계 조건
3. test_snake_move_null - NULL 입력"
`}
