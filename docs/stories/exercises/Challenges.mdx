1. **단계별 접근**: 큰 기능을 작은 단계로 나누세요
2. **테스트 작성**: 기능 구현 전에 테스트 케이스를 먼저 작성하세요
3. **Agent 활용**: 막힐 때 Agent에게 힌트를 요청하세요
