## Challenge 8: 저장/불러오기 기능

### 목표
게임 상태를 파일로 저장하고 불러오는 기능을 구현합니다.

### 요구사항
- S 키로 현재 상태 저장
- L 키로 저장된 상태 불러오기
- 저장 파일 형식: 바이너리 또는 JSON

### 힌트
```
AI Agent에게:
"save_load.c 모듈을 만들어서 게임 상태를
 save.dat 파일로 저장하고 불러오는 기능을 구현해줘"
```

### 고려사항
- 뱀의 전체 몸통 위치 저장
- 음식 위치, 점수, 방향 저장
- 에러 처리 (파일 없음, 읽기 실패 등)

## Challenge 9: AI 자동 플레이

### 목표
AI가 자동으로 게임을 플레이하는 모드를 추가합니다.

### 요구사항
- A 키로 AI 모드 토글
- 기본 전략: 음식을 향해 이동
- 벽과 자기 자신 충돌 회피

### 힌트
```
AI Agent에게:
"ai_player.c 모듈을 만들어서 간단한 AI가
 음식을 먹으러 가도록 구현해줘.
 BFS나 간단한 휴리스틱을 사용해도 좋아"
```

### 알고리즘 옵션
1. Greedy: 가장 가까운 방향으로
2. BFS: 최단 경로 탐색
3. A*: 휴리스틱 최적화

## Challenge 10: 멀티플레이어

### 목표
2인용 대전 모드를 구현합니다.

### 요구사항
- Player 1: WASD 키
- Player 2: 방향키
- 각 플레이어별 점수 표시
- 다른 플레이어와 충돌 시 게임 오버

### 힌트
```
AI Agent에게:
"2인용 게임을 위해 Snake 배열을 관리하고,
 각 플레이어별 입력을 처리하도록 수정해줘"
```

### 아키텍처 변경
- `Game` 구조체에 `Snake players[2]` 추가
- 입력 처리 분리
- 렌더링 시 두 뱀 모두 표시
