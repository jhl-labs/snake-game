# Challenge: Beginner Level

초급 난이도 도전 과제입니다.

---

## Challenge 1: 점수 시스템 추가

### 목표
게임에 점수 시스템을 추가합니다.

### 요구사항
- 음식을 먹을 때마다 10점 추가
- 현재 점수를 화면에 표시
- 게임 오버 시 최종 점수 표시

### 힌트
```
AI Agent에게:
"game.h에 점수를 저장할 변수를 추가하고,
 음식을 먹을 때 점수가 증가하도록 해줘"
```

### 예상 변경 파일
- `include/game.h` - 점수 변수 추가
- `src/game.c` - 점수 로직 추가
- `src/renderer.c` - 점수 표시

## Challenge 2: 게임 속도 조절

### 목표
게임 속도를 조절하는 기능을 추가합니다.

### 요구사항
- 기본 속도 설정
- 점수가 높아질수록 속도 증가
- 최대 속도 제한

### 힌트
```
AI Agent에게:
"게임 속도를 조절하는 기능을 추가해줘.
 점수가 50점 올라갈 때마다 속도가 10% 빨라지게 해줘"
```
