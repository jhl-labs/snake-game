# Exercises - 실습 과제

이 디렉토리는 AI Agent 교육을 위한 실습 파일들을 포함합니다.

## 파일 목록

| 파일 | 용도 | 난이도 |
|------|------|--------|
| `buggy_snake.c` | 버그 수정 실습 | 초급~중급 |
| `test_example.c` | TC 작성 실습 | 초급 |
| `refactor_before.c` | 리팩토링 실습 | 중급 |
| `challenge_*.md` | 도전 과제 | 다양 |

## 사용 방법

### 1. 버그 수정 실습
```bash
# Cursor 또는 Claude Code에서
> "exercises/buggy_snake.c 파일의 버그를 찾아서 수정해줘"
```

### 2. TC 작성 실습
```bash
> "exercises/test_example.c를 참고해서 snake_grow 함수의 테스트를 작성해줘"
```

### 3. 리팩토링 실습
```bash
> "exercises/refactor_before.c를 프로젝트 코딩 스타일에 맞게 리팩토링해줘"
```

## 정답 확인

각 실습의 모범 답안은 `solutions/` 디렉토리에 있습니다.
(교육자용 - 수강생에게는 비공개)
